定位原理

首先我们来了解一下，什么锚点？

在海上，水手们会将锚丢入海中，或者靠岸时将锚抛上岸，其固定目的是让船固定位置。而锚点(anchor)也是一样，只不过场景不同罢了。

页面锚点定位可以通过name或者id属性来定位。name已经被HTML5废弃了，就不再多提。

在页面上，每个id都会自动创建一个锚点，同时还会生成一个hash，表示所处的文档位置。浏览器可以通过hash来确定位置，使用的方式一般如下：
